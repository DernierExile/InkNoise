# InkNoise sample production pack

This pack shows the kind of deliverable InkNoise Studio is built around: input images, processed outputs and a reusable recipe.

Files:

- `input/campaign-01-editorial.jpg` - original campaign image
- `input/campaign-02-food.jpg` - original campaign image
- `output/campaign-01-editorial-inknoise.png` - processed InkNoise-style output
- `output/campaign-02-food-inknoise.png` - processed InkNoise-style output
- `inknoise-production-recipe.json` - portable recipe sample

In real production, Studio/Founder exports are intended for clean 4K files, reusable recipes and batch ZIP delivery.
